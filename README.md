# Drip Irrigation Design App (Cordova)

Android app that automates design calculations for drip irrigation systems.
link to google drive; https://drive.google.com/file/d/1OaHQgOFcNnXz7dy8PcJlWDiU_pI2GGJ7/view?usp=sharing

## What It Does
- Select crop type & irrigation needs
- helps to select appropriate:
  - Dripper discharge
  - Number of drippers per plant
  - Plant spacing
  - Daily irrigation plots
  - Pressure losses
  - Pipe sizes (main, submain, and drip lines)
- Determines if the system will function properly

## Tools
Cordova, JavaScript, HTML/CSS
